การแนะนำซอฟต์แวร์

แกดเจ็ตเลียนแบบเว็บไซต์เป็นเครื่องมือสำหรับการดาวน์โหลดหน้าเว็บแบบคงที่ผ่าน URL ดาวน์โหลดโค้ด html จาก URL อินพุต แยก JS, Css, รูปภาพ, รูปภาพ, Flash และ URL ไฟล์สแตติกอื่นๆ จากนั้นแยก URL ไฟล์สแตติก Image ออกจากโค้ด Css ที่ดาวน์โหลด ดาวน์โหลดไฟล์คงที่ผ่าน URL และตั้งค่า ตามกฎการบันทึกของซอฟต์แวร์ แก้ไขพาธลิงก์โค้ด html และ css โดยอัตโนมัติ และสุดท้ายไฟล์สแตติกเหล่านี้จะถูกบันทึกลงในโฟลเดอร์คอมพิวเตอร์ตามหมวดหมู่

แกดเจ็ตเว็บไซต์เลียนแบบ v9.0

  • แก้ไขปัญหาการเปลี่ยนชื่อไฟล์เดียวกัน
  • เพิ่มอุปกรณ์จำลองแบบกำหนดเอง
  • เพิ่มความสามารถในการใช้ใบรับรอง https
  • แก้ไขปัญหาที่การกำหนดค่าไม่สามารถบันทึกได้
  • ปรับฟังก์ชันการเข้ารหัสการรับรู้อัตโนมัติให้เหมาะสม
  • เพิ่มประสิทธิภาพการดาวน์โหลด
  • คุณสามารถวางอินพุตเพื่อแก้ไขที่อยู่บันทึกได้
  • เพิ่มทางลัดการเลือกกล่องข้อความ Ctrl+A
