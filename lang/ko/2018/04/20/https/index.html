HTTPS URL을 다운로드할 때 원격 서버가 다음 오류를 반환합니다: (400) Bad Request Error

By admin

2018-04-20에 게시됨

기사 디렉토리

HTTPS URL을 다운로드할 때 원격 서버가 (400) 잘못된 요청 오류라는 오류를 반환합니다.
모방 웹사이트 가젯이 HTTPS URL을 다운로드할 때 "원격 서버에서 오류(400) 잘못된 요청을 반환했습니다."라는 오류가 보고됩니다.

이유: 컴퓨터에 https 인증서가 설치되어 있지 않습니다.

해결 방법: IE 브라우저를 사용하여 웹사이트에 접속하십시오. IE 브라우저를 통해 접속해야 페이지가 표시됩니다.

다른 문제:
여러 URL을 다운로드하려면 해당 URL을 모두 방문해야 합니까?
동일한 도메인 이름으로 하나의 URL에 액세스하면 됩니다.
