【업데이트】웹사이트 모방 가젯 기록 업데이트

By admin

2015년 6월 27일에 게시됨

기사 디렉토리

** 다운로드 링크: ** Baidu Cloud Disk의 모든 버전

** 버전 8.2 **

1. [*] URL 목록 수정, 삭제 시 선택이 안되는 문제를 수정하였습니다.

** 버전 8.1 **

1. [*] 잘못된 URI 수정: URI 문자열이 너무 깁니다.

** 버전 8.0 베타 **

1. [+] 폰트 아이콘 다운로드 기능이 추가되었습니다.
2. [+] 위챗 홈페이지 다운로드 기능이 추가되었습니다.
3. [+] 주소를 저장하는 것을 잊지 마세요.
4.[#]URL 공간을 다운로드할 수 없는 문제를 수정했습니다.
5. [#]다른 디렉터리에 동일한 파일 이름을 덮어쓰는 문제를 수정합니다.
6. [#]로그인 정보를 얻는 기존 Chrome 및 IE 방법을 제거하고 사용자가 로그인 정보를 입력하는 방식으로 대체합니다.
7. [#] .Net 프레임워크 버전을 줄이고 .Net4.0 버전만 설치하면 됩니다.

** 버전 7.0 **

1. 압축된 전송 파일을 자동으로 풀어서 저장합니다.
2. 원래 내장된 IE 브라우저를 취소합니다.
3. Chrome 로그인 페이지 다운로드 방법을 추가했습니다(회원 관리 백엔드 또는 관리자 백엔드 페이지 다운로드). 기존 IE 로그인 다운로드 방법은 변경되지 않습니다.
4. 메뉴 표시줄의 새 웹사이트는 컴퓨터 단말기와 모바일 단말기로 구분됩니다. 기본 열기 소프트웨어는 컴퓨터 단말기입니다.

** 버전 6.0 **

1. 에뮬레이터 오류 문제를 수정합니다.
2. 로그인 페이지에서 다운로드 실패 문제를 수정했습니다.
3. 중복 링크 처리 오류를 수정합니다.

** 버전 5.0 **

1. 주로 휴대폰이나 태블릿 등 모바일 사용자만 접속할 수 있는 웹사이트에 모바일 시뮬레이터를 추가하세요.
2. 타이머를 추가하세요.
3. 크기 조정 후 Windows 글꼴이 불완전하게 표시되는 문제, 프롬프트 표시줄 표시 위치가 어지러워지는 문제를 수정했습니다.
4. 다시 다운로드할 때 중복된 파일이 나타나는 문제를 수정했습니다.
5. 컬렉션 읽기 및 쓰기의 스레드 안전 문제를 수정합니다.
6. 코드를 최적화하고 더 나은 경험을 제공하십시오.

** 버전 4.0 **

1. 여러 URL을 동시에 다운로드할 수 있습니다.
2. 동일한 CSS 파일을 반복적으로 다운로드하는 문제를 수정했습니다.
3. base64 이미지 처리 오류를 수정했습니다.
4. 다음 내용이 포함된 페이지를 수정하세요.태그가 링크 주소를 올바르게 얻을 수 없습니다.
5. 동적 JS 및 CSS 파일의 경우 확장자 이름을 수정합니다.
6. 데이터 저장 방식을 수정합니다.
7. 선택적 웹 페이지 인코딩을 제거하고 시스템을 사용하여 페이지 인코딩을 자동으로 식별합니다.
8. 탐지 URL을 최적화하고 잘못된 URL을 더 빠르게 탐지합니다.
9. 데이터 처리 결과를 분류하고 표시하기 위한 별도의 시스템 설정.
10. 이스케이프 문자 /#/about:blank 및 기타 처리가 포함된 링크를 추가합니다.
11. 콘텐츠를 보다 정확하게 찾고 식별할 수 있도록 내부 작업 내결함성 제어를 강화합니다.
12. 스타일 파일을 HTML 페이지로 가져오는 처리가 추가되었습니다.
13. 소프트웨어 인터페이스를 아름답게 만드십시오.

** 버전 3.0 **

1. 로그인 페이지 다운로드 기능을 추가합니다.
2. 소프트웨어 구성을 기억하는 기능을 추가합니다.
3. 사용자 정의 HTML 파일 이름을 추가합니다.
4. 다운로드한 파일이 WebSite 폴더에 저장되지 않고 도메인 이름을 딴 폴더에 저장되도록 수정합니다.
5. 동일한 웹사이트의 여러 페이지를 차례로 다운로드할 수 있도록 추가했습니다(Css, Js, 이미지 및 기타 파일은 반복적으로 다운로드되지 않습니다).
추신: 루트 디렉터리에 생성된 Config.xml은 다운로드가 완료될 때마다 자동으로 생성되는 소프트웨어의 구성 파일이며, log.Dat 파일은 도메인 이름 디렉터리에 생성됩니다. 현재 다운로드된 파일을 기록하는 로그 파일입니다.

** 버전 2.3 **

1. 일련의 사소한 버그를 수정했습니다.

** 버전 2.2 **

1. 버전 V2.0 이후 계속해서 충돌 문제를 해결합니다(이전 버전에서는 완전히 해결되지 않았습니다).

** 버전 2.1 **

1. 일부 URL에서 코드를 획득할 수 없는 문제를 수정했습니다.
2. Net 프레임워크 버전으로 인한 충돌 문제를 해결했습니다.

** 버전 2.0 **

1. 디렉토리 변경 기능을 추가합니다.
2. 다운로드 실패 파일을 무한 대기하는 문제(멈추는 문제)를 수정했으며, 다운로드한 파일에 대해 시간 제한을 설정할 수 있습니다.
3. gzip 및 deflate 압축에 대한 지원을 추가합니다.
4. 페이지 배경 속성 배경 이미지에 대한 지원을 추가합니다.
5. base가 포함된 페이지에서 href 속성 값을 수정합니다.
6. 성능을 최적화합니다.
** 버전 1.1 **
멀티라인 모드에서 사진 및 플래시 파일을 제대로 다운로드할 수 없는 문제를 수정했습니다.

** 버전 1.0 **

첫 번째 릴리스 버전.
