Webサイトのエンコーディングを取得する方法

1. Chrome ブラウザ/Firefox ブラウザ: 右クリック -> Web ページのソース コードを表示 -> 「charset」を検索します。 2. IE ブラウザ: 右クリック -> ソースの表示 -> 「charset」を検索します。 Web ページのコーディングには 2 つの方法があります (参考文書: http://w3schools.fzxgj.top/html/)
