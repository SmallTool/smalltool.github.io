При загрузке URL-адреса HTTPS удаленный сервер возвращает ошибку: (400) Ошибка неверного запроса. При загрузке URL-адреса HTTPS гаджет-имитатор веб-сайта сообщает: «Удаленный сервер возвращает ошибку: (400) Неверный запрос. Причина ошибки». : На вашем компьютере не установлен сертификат https. Решение: используйте браузер IE для доступа к веб-сайту. Доступ к нему должен осуществляться через браузер IE, и страница будет отображена.
