HTTPS URL တစ်ခုကို ဒေါင်းလုဒ်လုပ်သောအခါ၊ အဝေးထိန်းဆာဗာသည် အမှားတစ်ခု ပြန်ပို့ပေးသည်- (400) ဆိုးရွားသော တောင်းဆိုမှု အမှား

HTTPS URL တစ်ခုကို ဒေါင်းလုဒ်လုပ်သောအခါ၊ အဝေးထိန်းဆာဗာမှ အမှားတစ်ခု ပြန်ပေးသည်- (400) ဆိုးရွားသော တောင်းဆိုချက် အမှားတစ်ခု HTTPS URL ကို ဒေါင်းလုဒ်လုပ်သောအခါ၊ အတုအယောင် ဝဘ်ဆိုဒ် ဂက်ဂျက်က "အဝေးထိန်းဆာဗာသည် အမှားတစ်ခု ပြန်ပေးသည်- (400) မှားယွင်းသော တောင်းဆိုချက်" : သင့်ကွန်ပြူတာတွင် https လက်မှတ် ထည့်သွင်းထားခြင်းမရှိပါ။ ဖြေရှင်းချက်- ဝဘ်ဆိုက်ကိုဝင်ရောက်ရန် IE ဘရောက်ဆာကို အသုံးပြု၍ ၎င်းကို IE ဘရောက်ဆာမှတစ်ဆင့် ဝင်ရောက်ကြည့်ရှုရမည်ဖြစ်ကာ စာမျက်နှာကို ပြသမည်ဖြစ်သည်။
