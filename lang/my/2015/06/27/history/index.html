** ဗားရှင်း 7.0 **

1. အလိုအလျောက် ချုံ့ပြီး ချုံ့ထားသော ဂီယာဖိုင်များကို သိမ်းဆည်းပါ။
2. မူရင်း built-in IE ဘရောက်ဆာကို ပယ်ဖျက်ပါ။
3. Chrome လော့ဂ်အင်စာမျက်နှာ ဒေါင်းလုဒ်နည်းလမ်းကို ပေါင်းထည့်ခဲ့သည် (အသင်းဝင်စီမံခန့်ခွဲမှု နောက်ခံဖိုင် သို့မဟုတ် မန်နေဂျာ ကျောထောက်နောက်ခံ စာမျက်နှာကို ဒေါင်းလုဒ်လုပ်ပါ) မူရင်း IE လော့ဂ်အင် ဒေါင်းလုဒ်နည်းလမ်းသည် မပြောင်းလဲပါ။
4. မီနူးဘားရှိ ဝဘ်ဆိုက်အသစ်ကို ကွန်ပျူတာ ဂိတ်နှင့် မိုဘိုင်းဂိတ်ဟူ၍ ပိုင်းခြားထားသည်။ မူရင်းဖွင့်ဆော့ဖ်ဝဲလ်မှာ ကွန်ပျူတာ တာမင်နယ်ဖြစ်သည်။

** ဗားရှင်း 6.0 **

1. emulator ချို့ယွင်းမှုပြဿနာကို ဖြေရှင်းပါ။

3. ပွားနေသောလင့်ခ်များကို လုပ်ဆောင်ရာတွင် အမှားကိုပြင်ပါ။

** ဗားရှင်း 5.0 **

1. အဓိကအားဖြင့် မိုဘိုင်းလ်ဖုန်း သို့မဟုတ် တက်ဘလက်များကဲ့သို့ မိုဘိုင်းအသုံးပြုသူများသာ ဝင်ရောက်နိုင်သော ဝဘ်ဆိုက်များအတွက် မိုဘိုင်းလ် simulator တစ်ခုကို ထည့်ပါ။
2. အချိန်တိုင်းကိရိယာတစ်ခုထည့်ပါ။
3. ချဲ့ထွင်ပြီးနောက် Windows ဖောင့်များ၏ မပြည့်စုံသောပြသမှုပြဿနာနှင့် prompt bar များ၏ ပြကွက်များ မမှန်သည့်အနေအထားပြဿနာကို ဖြေရှင်းပေးသည်။
4. ထပ်မံဒေါင်းလုဒ်လုပ်သည့်အခါ ပေါ်လာသော မိတ္တူဖိုင်များကို ပြင်ဆင်ပါ။
5. စုစည်းဖတ်ရှုခြင်းနှင့် ရေးသားခြင်းဆိုင်ရာ ချည်ကြိုးဘေးကင်းရေးပြဿနာကို ဖြေရှင်းပါ။
6. ကုဒ်ကို ပိုမိုကောင်းမွန်အောင်ပြုလုပ်ပြီး ပိုမိုကောင်းမွန်သော အတွေ့အကြုံကို ပေးဆောင်ပါ။

** ဗားရှင်း 4.0 **

1. URL အများအပြားကို တပြိုင်နက်ဒေါင်းလုဒ်လုပ်ခြင်းကို အောင်မြင်အောင်လုပ်ပါ။
2. တူညီသော CSS ဖိုင်ကို ထပ်ခါတလဲလဲ ဒေါင်းလုဒ်လုပ်ခြင်း ပြဿနာကို ဖြေရှင်းပါ။
3. base64 ရုပ်ပုံလုပ်ဆောင်ခြင်းဆိုင်ရာ အမှားအယွင်းများကို ပြင်ဆင်ပါ။
4. ပါဝင်သော စာမျက်နှာကို ပြင်ပါ။တဂ်သည် လင့်ခ်လိပ်စာကို မှန်ကန်စွာမရနိုင်ပါ။
5. ပြောင်းလဲနေသော JS နှင့် CSS ဖိုင်များအတွက်၊ ၎င်းတို့၏ တိုးချဲ့အမည်များကို ပြင်ပါ။
6. ဒေတာသိမ်းဆည်းနည်းကို မွမ်းမံပါ။
7. ရွေးချယ်နိုင်သော ဝဘ်စာမျက်နှာကုဒ်ကို ဖယ်ရှားပြီး စာမျက်နှာကုဒ်ကို အလိုအလျောက် ခွဲခြားသတ်မှတ်ရန် စနစ်အား အသုံးပြုပါ။
8. ထောက်လှမ်းခြင်း URL ကို ပိုမိုကောင်းမွန်အောင်ပြုလုပ်ပြီး မမှန်ကန်သော URL များကို ပိုမိုမြန်ဆန်စွာ ရှာဖွေပါ။
9. ဒေတာလုပ်ဆောင်ခြင်းရလဒ်များကို အမျိုးအစားခွဲခြားပြီး ပြသရန် သီးခြားစနစ်ဆက်တင်များ။
10. Escape စာလုံးများပါရှိသော လင့်ခ်များကို ပေါင်းထည့်ပါ /#/about:blank နှင့် အခြားလုပ်ဆောင်ခြင်း။
11. အကြောင်းအရာကို ပိုမိုတိကျစွာ ရှာဖွေဖော်ထုတ်နိုင်စေရန် အတွင်းပိုင်းလုပ်ဆောင်ချက် အမှားခံနိုင်မှု ထိန်းချုပ်မှုကို မြှင့်တင်ပါ။
12. HTML စာမျက်နှာများသို့ စတိုင်ဖိုင်များကို တင်သွင်းရန် စီမံဆောင်ရွက်ပေးမှုကို ပေါင်းထည့်သည်။
13. software interface ကို လှပအောင်ပြုလုပ်ပါ။

** ဗားရှင်း 3.0 **

1. လော့ဂ်အင်စာမျက်နှာကို ဒေါင်းလုဒ်လုပ်ခြင်း၏ လုပ်ဆောင်ချက်ကို ပေါင်းထည့်ပါ။
2. ဆော့ဖ်ဝဲဖွဲ့စည်းမှုပုံစံကို မှတ်သားခြင်း၏ လုပ်ဆောင်ချက်ကို ပေါင်းထည့်ပါ။
3. စိတ်ကြိုက် HTML ဖိုင်အမည်ကို ထည့်ပါ။
4. ဒေါင်းလုဒ်လုပ်ထားသောဖိုင်များကို ဝဘ်ဆိုက်ဖိုင်တွဲတွင် သိမ်းဆည်းခြင်းမှ ဒိုမိန်းအမည်ကို ပေးထားသော ဖိုင်တွဲတစ်ခုတွင် သိမ်းဆည်းခြင်းသို့ ပြောင်းလဲပါ။
5. တူညီသောဝဘ်ဆိုက်၏ မတူညီသောစာမျက်နှာများကို တစ်ခုပြီးတစ်ခုဒေါင်းလုဒ်လုပ်နိုင်သည် (Css၊ Js၊ ရုပ်ပုံနှင့် အခြားဖိုင်များကို ထပ်ခါတလဲလဲဒေါင်းလုဒ်လုပ်မည်မဟုတ်ပါ) ထပ်လောင်းထည့်ပါ။
PS- ဒေါင်းလုဒ်လုပ်ပြီးသည်နှင့်၊ မှတ်တမ်းတစ်ခုဖြစ်သည့် ဒိုမိန်းအမည်လမ်းညွှန်တွင် log.Dat ဖိုင်ကို PS- PS- ဒေါင်းလုဒ်လုပ်တိုင်း အလိုအလျောက်ထုတ်ပေးသည့် ဆော့ဖ်ဝဲဖွဲ့စည်းပုံဖိုင်ဖိုင်ဖြစ်သည်။ လက်ရှိဒေါင်းလုဒ်လုပ်ထားသော ဖိုင်များကို မှတ်တမ်းတင်သည့် ဖိုင်။
